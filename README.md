视觉层次：重要内容使用视觉焦点设计

使用说明

直接打开index.html文件即可在浏览器中运行网站

所有资源文件需保持相对路径不变

如需修改样式，请编辑css/1.css文件

更新图片资源请替换images/目录中的对应文件
